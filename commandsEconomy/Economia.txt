:cyclone: Aplicación
 - En el momento de aplicar el nuevo sistema de economia, a cada usuario se le regresará el dinero de su caja fuerte en el caso de que tenga. Luego a ***todos*** los usuarios se les quitará un 30% de su dinero y se agregará al banco.

:cyclone: Reglas
- Estado neutro 240M 260M
 - No se aumentará ni disminuirá el dinero total del server a no ser que sea caso de emergencia (muchos usuarios usandolo / pocos usuarios usandolo).
 - Cada mes se cobrarán automáticamente un impuesto para los roles de club, entre 20k y 60k dependiendo de la cantidad de clubes a los que estés (esperanzas no cuenta), haciendo que el dinero recaudado vaya al banco.
 - Comprar una membresía costará 20k
 - No se podrá ver el total de dinero del banco, para que no jodan con los juegos
 - Tema work (horas y porcentajes pueden variar) =>
    - Si el banco tiene menos de 180M, no dará dinero
    - Si el banco tiene entre 180M y 220M, dará entre 1.000 y 2.000
    - Si el banco tiene entre 220M y 240M, dará entre 2.000 y 4.000
    - Si el banco tiene entre 240M y 260M, dará entre 4.000 y 6.000
    - Si el banco tiene entre 260M y 300M, dará entre 6.000 y 8.000
    - Si el banco tiene más de 300M, dará entre 8.000 y 10.000
    - Si el usuario tiene más de 3M, el work le dará el 50%

    Todos los usuarios: Un work diario el cual se podrá hacer cada 0:00h argentina
    Twitch Subscriber: Un work extra que se podrá hacer a partir de las 12:00h argentina 
    Nitro Booster: Dos work extra que se podrán hacer a las 6:00h y a las 18:00h argentina
        En caso de ser sub y nitro, podrás hacer los 4 works
